## How to use
```
/start - Start the bot
/help - get this message.
/track {Product Link} - Add product to tracking list.
/stop {Tracking ID} - Stop tracking.
/list - Get list of products that are being tracked.
```
-----------

## NOTE
```
- Do not clone this repo (You can fork it instead).

```
